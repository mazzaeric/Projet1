__call( string $name, array $args ) : mixed

Call method, any one

This magic method will be called when you're using any assertion and some method inside it, for example:

// inside your instance of phpRack_Test:
$this->assert->php->extensions->isLoaded('simplexml');

The call in the example will lead you to this method, and will call __call('simplexml', array()).
