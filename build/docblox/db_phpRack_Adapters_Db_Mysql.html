_parseJdbcUrl( string $url ) : array
inherited

Parse JDBC URL and return its components

Inherited from: \phpRack_Adapters_Db_Abstract::_parseJdbcUrl()

This method matches URLSs like:

jdbc:mysql://localhost:3306/test?username=login&password=password
jdbc:mysql://localhost:3306/test
jdbc:mysql://localhost:3306
jdbc:mysql://localhost

Mandatory parts of the URL are: "adapter", "host". All other params are optional and could be omitted.
