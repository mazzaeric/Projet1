Package: Tests

View in order to render test presentation page

This class is used as a view component in our simple MVC pattern, where model is {@link phpRack_Runner}, view is {@link phpRack_View} and controller is a simple script in {@link bootstrap.php}. An instance of this class is accepting variables and a script name in order to render it, for example:

$view = new phpRack_View();
$view->assign(array('name' => 'My Name To Render'));
$html = $view->render('index.phtml');

In this example, you can access "name" inside "index.phtml" like this: $this->name.
